AVATAR WORLD WAR
Avatar World War merupakan suatu game berbasis gcc yang dijalankan lewat terminal. Game ini bersifat turn-based strategy game dengan memanfaatkan gui yang terdapat pada command line interface linux.

GETTING STARTED
Untuk menjalankan game ini, silahkan mengikuti langkah - langkah berikut :

>Prerequisted
1. Terminal berbasis linux
2. gcc (recommended version 5.4.0)
3. git (optional)


>Installing / getting the game
Clone /  download the repo to your local machine : 
Tuliskan command berikut di terminal / git bash anda :
git clone https://github.com/William9923/Alstrukdat.git


>Lalu di folder / di file path dimana anda melakukan clone terhadap repo diatas,
ketikkan hal berikut di linux terminal yang telah terinstall gcc :
gcc main.c -o main -lm

>Untuk menjalankan game, silahkan tulis :
./main
